# openGL

git clone --recursive https://github.com/noah7288/openGL.git


## DEPENDENCES

- LINUX

    -   CMake minimum version 3.18

    -   Python version 3

    -   libgl1-mesa-dev

    -   librandr-dev

    -   libxinerama-dev

    -   libcursor-dev

    -   libxi-dev

    -   gdb

- WINDOWS

    -  CMake minimum version 3.18

    -  Python version 3

    -  Troubleshoot for msvcp140.dll or vcruntime140.dll is missing error
    -   -   https://docs.microsoft.com/en-US/cpp/windows/latest-supported-vc-redist?view=msvc-170
